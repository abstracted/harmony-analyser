harnony-analyser

harmony-analyser is a tool for harmonic analysis, featuring chord transition tool for MIDI input and audio analysis from WAVE input. The application was developed as a part of Master's thesis on Comenius University in Bratislava, Faculty of Mathematics, Physics and Informatics.
